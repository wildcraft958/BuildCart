ZIKSIR – MERN Intern Challenge
Task Title: BuildCart – A Full-Stack MERN Commerce Experience
Level: Intern

Task Description
Build a basic e-commerce web application using the MERN stack (MongoDB, Express.js, React.js,
Node.js). The application should allow users to register, log in, browse products, add products to a
cart, and place an order. An Admin role should be able to manage products and view all user
orders.

Core Requirements
1. Authentication & Roles
• User registration and login using JWT authentication.
• Two roles: User and Admin.
• Role-based access control for APIs and UI.
2. Product Management (Admin)
• Add a product with name, price, description, image URL, and stock quantity.
• Delete products.
• View all products.
3. Cart Functionality (User)
• Add products to cart.
• Update product quantity.
• Remove products from cart.
• View total cart price.
4. Orders
• Users can place an order from their cart.
• Store order details including user, products, total amount, and order date.
• Admin can view all users’ orders.
5. Frontend (React)
• Pages: Login, Register, Product Listing, Cart.
• Conditional UI based on role (Admin/User).
• Basic responsive layout and form validation.
6. Backend (Node.js & Express)
• REST APIs for authentication, products, cart, and orders.
• JWT-protected routes.
• Admin-only middleware for restricted routes.
7. Database (MongoDB)
• User schema: name, email, password, role.
• Product schema.
• Order schema.

Bonus (Optional)

• Product search or pagination.
• Persist cart after refresh.
• Deployment of the application.

Expected Deliverables
• GitHub repository link.
• README with setup instructions and API details.
• Sample admin credentials.
